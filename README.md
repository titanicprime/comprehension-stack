# Comprehension Stack

The Comprehension Stack is a unified map of the substrate-level systems designed to restore clarity, trust, and agency in high-entropy environments.

This repository explains the architecture, layers, invariants, and roadmap that connect the following projects:

- **Comprehension Scope** — human-level sensemaking protocol  
- **Clarity Index** — shared knowledge substrate  
- **BitRep Core** — attestation and trust substrate  
- **Truth–Freedom–Agency Protocol** — minimal governance substrate  
- **Moltbot Safe** — hardened agent behavior  

Each project is independent, but together they form a coherent civilizational stack.
